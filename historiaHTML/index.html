HISTORIA HTML

El origen de html se dio en el año 1980, cuando el fisico Tim Bernes-lee lo inventa con el fin de compartiR articulos cientificos. Para la primera versión se contaba con 18 etiquetas para texto y tambien dejaba enlazar varios documentos Usando hipertexto (comunicarse a travez de internet), para visualizarse en un navegador llamado world wide web (www).

ETIQUETAS HTML

*body: para el contenido.
*head: para información sobre el documento. *
*div: división dentro del contenido.
*a: para enlaces.
*strong: para poner el texto en negrita.
*br: para saltos de línea.
*H1-H6: para el tamaño de títulos dentro del contenido.
*img: para añadir imágenes al documento.
*ol:para listas ordenadas.
*ul: para listas desordenadas.
*li:para elementos dentro de la lista.
*p:para parágrafos

CARACTERISTICAS PRINCIPALES DE HTML

 1.Es facil de usar y entender.
2.Es utilizado para crear páginas web.
3.Permite escribir hipertexto.
4.Se puede acceder desde cualquier dispositivo.
5.Tiene un despliegue rápido.
6.Es reconocido y admitido por cualquier tipo de explorador web
