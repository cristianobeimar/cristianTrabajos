Manejo de DOM

DOM vasico

DOM (Document Object Model) es una interfaz de programación que permite a los scripts actualizar el contenido, la estructura y el estilo de un documento mientras este se está visualizando.

  • Element.innerHTML: Cambia el contenido HTML de un elemento.
  • Element.textContent: Cambia el contenido de texto de un elemento.
  • Element.setAttribute(name, value):Establece el valor de un atributo.
  • Element.getAttribute(name):Obtiene el valor de un atributo.
